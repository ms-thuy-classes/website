Không reload trang. --- # Dữ liệu bài viết Tất cả metadata lưu trong ``` data/articles.json ``` Ví dụ: ```json [ { "id": "001", "title": "100 Từ vựng IELTS Topic Education", "category": "IELTS", "filter": "Vocabulary", "thumbnail": "assets/images/ielts1.jpg", "description": "Tổng hợp từ vựng IELTS chủ đề Education.", "type": "local", "link": "data/post-001.html", "date": "2026-07-09" }, { "id": "002", "title": "50 Cấu trúc ngữ pháp TOEIC", "category": "TOEIC", "filter": "Grammar", "thumbnail": "assets/images/toeic.jpg", "description": "Ngữ pháp TOEIC thường gặp.", "type": "external", "link": "https://example.com/toeic-grammar", "date": "2026-07-10" } ] ``` --- # Quy tắc mở bài viết Nếu ``` type = local ``` thì khi click sẽ mở ``` data/post-001.html ``` Nếu ``` type = external ``` thì mở đúng URL ngoài trong tab mới. Ví dụ: ``` https://abc.com/... ``` --- # Trang danh sách bài viết Hiển thị dạng Grid.
